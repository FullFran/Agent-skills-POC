# Herramientas MCP Disponibles

## Context7 Docs
Servicio de consulta de documentación técnica para librerías y frameworks.

- **resolve-library-id**: Resuelve el nombre de un paquete a un ID de librería compatible con Context7.
  - Args: `libraryName: string`, `query: string`
- **query-docs**: Recupera documentación y ejemplos de código actualizados.
  - Args: `libraryId: string`, `query: string`
